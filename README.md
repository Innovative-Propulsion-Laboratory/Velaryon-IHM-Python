This Velaryon Human–Machine Interface was developed in Python, primarily using the PyQt5 library. Although the code is functional, it was not fully optimized, mainly because development later shifted toward a Qt/C++ implementation. As a result, no further optimization was carried out on this Python version.

Update: Following the demonstrated efficiency of the C++ version of the interface, along with the many optimizations implemented on the DAQ and control systems, this Python version was ultimately never used for engine operations. However, it may still serve as a useful reference for future Python-based interfaces developed with PyQt.
