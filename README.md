This Velaryon Human–Machine Interface was developed in Python, primarily using the PyQt5 library. Although functional in principle, the code was never thoroughly optimized and still lacks several features, as development will transitionned to a Qt/C++ implementation. As a result, no further optimization was carried out on this Python version.

Update: Following the demonstrated efficiency of the C++ version of the interface, along with the many optimizations implemented on the DAQ and control systems, this Python version was ultimately never used for engine operations . However, it may still serve as a reference for future PyQT-based interfaces.
